prs
===

This is a simple command-line implementation of the classic **paper-rock-scissors** game written in C.

Files
-----

* game.h
* main.c
* compute.c
* printing.c
* utility.c
* makefile

Requirements
------------

* gcc
* make

Compilation
-----------

To compile, run:

	make

To clean the compiled files, run:

	make clean
